OCP到SE 11的考试都考完，下一步该准备OCE的考试了。虽然其内容已经有不少在工作中反复使用，但从来也没系统的好好看过，正好利用这个机会查漏补缺。  

## 学习方针
去书店查看了一下，几乎没有直接面向OCE考试的书籍，介绍JavaEE7，JavaEE8的书也很少，多是入门。不过通过考试要点不难发现，Java EE的只是多是大的模块，那么不如结合实际的工作，对各个大块来进行有针对性的学习。

##  考试要点

### 1. Create Java Web Application using JSPs

1. Describe JSP life cycle
2. Describe JSP syntax, use tag libraries and Expression Language(EL)
3. Handle errors using Servlets and Java Server Pages
