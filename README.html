    XSUN 个人主页

    工具

    使用Hexo生成，hexo相关请参考hexo.io

    结构

    branch:

    1. master 网页部署用
2. develop 网页生成用的hexo工程文件

    文件

    * scaffolds 生成文章，草稿或者网站分页的模板
* source 资源文件夹，保存文章及引用到的代码或者图片
* themes 主题，为submodule，主题代码在其他repository保存。
* _config.yml 用来设置网站的各种参数，名称，连接，功能启用
* .gitignore 无视
* .gitmodules 定义submodule用
* package.json node包管理员
* package-lock.json 管理员陷饺子
