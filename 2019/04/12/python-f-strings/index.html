2019-04-12

原文： python-f-string

自3.6版本开始，python引入了一种叫做f-strings的写法，使用改写发可以极大地增强字符串的可读性（当然好处不止这些).
不过在学习f-strings之前，先看看以往的写法都有哪些。

旧方法

在3.6之前，我们常用的字符串格式化写法有两种：

  1. 使用转移符 %
  2. 使用字符串的格式化方法 str.format()
